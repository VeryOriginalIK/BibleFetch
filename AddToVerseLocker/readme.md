## Getting Started

### Prerequisites
a Verse locker account.
a bible.xml - (zefania format will work for sure, others may also work)

### How to run

1. Create a dotenv file (use .env.example),
2. If your bible is not in English, update BOOK_MAP to have the correct translations of the books of the Bible.
3. Install pip packages
   ```sh
   pip install selenium
   pip install dotenv
   ```
4. Run the program.
5. Complete the captcha. Then return to the terminal and press enter.
6. Everything else will be done automatically.
7. Suggestion: Create a folder for the collections which were created by the app,
